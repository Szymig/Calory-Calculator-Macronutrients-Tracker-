README
Application that helps you to track your calories and macronutrients throughout the day, and compare it with past consumption. 

Project created to learn basics of C#, Linqu and SQL (all stored procedures written manually) in form of somewhat usefull aplication.

Technologies used:
Visual Studio 2017, MSSQL Server Express 2014, Dapper (https://github.com/StackExchange/Dapper)
